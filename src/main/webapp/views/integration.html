Retailer Specific Settings

Retailer ID: 8727f8d6-566b-11e1-a45b-4040d4afcf5f

This is the id used to identify your retailer api calls to the system.

Domains

Production: http://www.echoed.com/

Test Environment: http://demo.echoed.com/

Integration

  1. Integration with Echoed requires the confirmation page to request the ‘Share’ button via the Echoed API. (Refer to “Echo Button Request Parameters below”) A button should be requested for each product on the confirmation page.
  2. Each painted button should open a window via javascript to the corresponding Echoed API echo call (Refer to ‘Echo Confirmation URI Parameters’). The javascript should be formatted as follows: window.open(<echoUrl>,’Echoed’,’width=960,height=500,toolbar=0,menubar=0,location=0,status=1,scrollbars=0,resizable=0,left=0,top=0’);
  3. Lightbox / Popup integration for the confirmation screen. This will cause a overlay to appear advertising the Echoed platform to the user upon check out. (Refer to “Echo Lightbox Integration”)

Echo Button Request Parameters

Production Url: http://www.echoed.com/echo/button

Test Environment URL : http://demo.echoed.com/echo/button

URL Parameters:

Parameter

Type

Description

retailerId

String

Unique Id provided by Echoed

customerId

String

Unique Customer Id used for your system. Used to match it back to your system.

productId

String

Unique Product Identification for the client’s system (SKU)

boughtOn

Date

Date of purchase

orderId

String

Order # of associated product

price

Float

Price of product

imageUrl

String

URL for the image used for the product

landingPageUrl

String

Desired URL that click throughs should direct users to. Ideally the product page for the item bought.

productName

String

Name of the Product

category

String

Product Category.

brand

String

Brand of the product.

description

String

Description of the product purchased

Note: All parameter values should be URI Encoded.

Echo Lightbox Integration

Append the following HTML markup to the bottom of the confirmation page. If the jquery library is already used in the page, it should not be included again.

HTML Markup:

<link rel=”stylesheet” type=”text/css” href=”http://v1-cdn.echoed.com/echoed-lightbox.css”/>

<div id=”echoed-popup”><img src=”http://v1-cdn.echoed.com/images/8727f8d6-566b-11e1-a45b-4040d4afcf5f.jpg”/></div>

<script type=”text/javascript” src=”http://v1-cdn.echoed.com/js/jquery-1.6.2.min.js”></script>

<script type=”text/javascript” src=”http://v1-cdn.echoed.com/js/jquery.echoed.lb.js”></script>

<script type=”text/javascript” src=”http://v1-cdn.echoed.com/js/shopify-echoed.js”></script>
