BigCommerce Integration

Enable API Access
To grant a user access to the API, go to your store's control panel and follow these steps:
  1. Login To Your BigCommerce Store
  2. Navigate to the Users area.
  3. Edit or Create the user you want to allow access to the API.
  4. Scroll down to the Enable the API setting and check the Yes, allow this user to use the API check box
  5. An access token will be generated and shown below
  6. Copy the API Path and the API Token
  7. Click the Save button

Finalize The Integration
Head back over to your BigCommerce store, you're almost done!

  2. Click on Settings and then select Conversions from the drop down menu
  3. In the Conversion Tracking Code Box, enter the code from your email confirmation
  4. Click Save
